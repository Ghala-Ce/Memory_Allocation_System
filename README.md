Memory Allocation Simulator
A C++ project that visualizes non-contiguous memory storage using a 1024-block array.
Key Feature: It maps file blocks to available memory spaces and displays the status as a visual grid.

حاكي توزيع الذاكرة
مشروع بلغة ++C يوضح آلية تخزين البيانات بشكل غير متصل (مرن) داخل مصفوفة مكونة من 1024 بلوك.
الميزة الأساسية: يقوم بتوزيع أجزاء الملف على المساحات المتاحة في الذاكرة ويعرض الحالة النهائية على شكل شبكة بصرية.
